# Redirects for The Wikipedia Library

This file can be imported into the REDIRECTOR add on for [Chrome](https://chrome.google.com/webstore/detail/redirector/ocgpenflpmgnfapjedencafcfakcekcd) and [Firefox](https://addons.mozilla.org/en-GB/firefox/addon/redirector/) and will redirect various journal URLs to the full text versions available through The Wikipedia Library.
